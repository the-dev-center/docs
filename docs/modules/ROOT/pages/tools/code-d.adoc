= code-d Documentation

`code-d` is a VS Code extension for D development. It provides features like syntax highlighting, code completion, and debugging support.

== Features

* **Syntax Highlighting**: Provides syntax highlighting for D language.
* **Code Completion**: Offers intelligent code completion.
* **Debugging**: Debug your D applications directly in VS Code.

== Installation

1. Open VS Code.
2. Go to Extensions (`Ctrl+Shift+X`).
3. Search for "code-d".
4. Install the extension.

== Configuration

You can configure `code-d` via VS Code settings. Look for `d.*` settings.

For more details, visit the link:https://github.com/Pure-D/code-d[official repository].
