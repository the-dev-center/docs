Knowledge Base Orientation

Welcome to the Dev Center Knowledge Base. This is the central repository for technical intelligence, best practices, and the collective memory of our engineering team.

App Purpose

The Dev Center application is designed to be the single source of truth for our development ecosystem. It centralizes project metadata, automates repetitive setup tasks, and provides a unified interface for managing developer identities across the services we use every day.

How to Use This Documentation

We follow the Diátaxis framework, which organizes documentation by the user’s intent and cognitive state. Choose the path that matches your current goal:

[.text-center] === 🎓 Tutorials Intent: Learning Focused on acquisition. Follow our step-by-step guides to build basic competence in our systems.

🚀 Start Onboarding

[.text-center] === 🛠️ How-to Guides Intent: Execution Focused on application. "I have a goal, show me the recipe." Task-oriented guides for real-world problems.

📂 Browse How-tos

[.text-center] === 📚 Reference Intent: Retrieval Focused on facts. Technical specifications, API documentation, and checklists.

📋 View Reference

[.text-center] === 🧠 Explanation Intent: Comprehension Focused on understanding. Discursive content that explains the "Why" behind our architecture and philosophy.

💡 Explore Concepts

Quick Start: If you are new to the team, start with the Onboarding Tutorial. It will guide you through setting up your machine using our Recommended Tools.
